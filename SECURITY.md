# Security Policy

## Supported Versions

Please use the latest version of ShieldMonkey. We only provide security updates for the latest stable release.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅ |
| < Latest| ❌ |

## Reporting a Vulnerability

**Please do not open a GitHub issue for security vulnerabilities.**

If you discover a security vulnerability, please report it via one of the following methods:

1. **GitHub Security Advisory**: Use the "Report a vulnerability" button in the Security tab of this repository.
2. **Contact maintainers**: If a private reporting tool is not available, please contact the project maintainers directly.

We will acknowledge your report within 48 hours and provide a detailed response within 1 week.

## Our Commitment

As a browser extension handling sensitive information, security is our top priority. We review dependencies regularly and use automated tools to scan for known vulnerabilities.
